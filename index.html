PHP tili Haqida qisqacha ma'lumot

  1. PHP server skript tili , dinamik va interaktiv veb-sahifalarni yaratish uchun juda qulay vosita hisoblanadi.
  2. PHP keng miqyosda qo'llaniladi, bepul va samarali. U Microsoftning ASP ga o'xshab raqobatchilar bilan teng kela oladi.
  3. PHP skriptlar serverlarda ishlaydi. Uni kompyuteringizda local server orqali ishlatishingiz mumkin.

Boshlashdan oldin nimalarni bilishingiz kerak?

PHP ni o'rganishdan oldin siz quyidagi dasturlash tillaridan asosiy tushunchalarga ega bo'lishingiz kerak bo'ladi:

  • HTML
  • CSS
  • JavaScript

PHP o'zi nima?

  • PHP ni ko'chirib olish va uni ishlatish mutlaqo bepul
  • PHP so'zi kengaytmaga ega bo'lib, u PHP: Gipermatnli Preprosessor*
  • PHP keng qo'llaniladi va u ochiq manbali skript til.
  • PHP skriptlar faqat serverlarda ishlaydi!

*Preprosessor - boshqa dasturning kiritilgan talablariga mos keladigan ma'lumotlarni o'zgartiradigan kompyuter dasturi

PHP juda ham qiziqarli va mashxur til hisoblanadi!

  • Veb tarmoqda blog tizimlarini (Wordpress) aynan PHP da yaratilishi uni mashxurlikka olib chiqdi.
  • Eng katta ijtimoiy tarmoqning PHP da yaratilishi (Facebook)
  • Yangi boshlovchilar uchun serverda tez ishlash imkoniyati bor, bu qulaylik keltiradi.

PHP fayli nima?

  • PHP fayllari matn, HTML, CSS, JavaScript va PHP kodlarini o'z ichiga olishi mumkin
  • PHP kodi serverda bajariladi va natija brauzerda oddiy HTML sifatida qaytariladi
  • PHP fayllari .php kengaytmasiga ega.

PHP nima qiloladi?

  • PHP dinamik sahifa kontentini yaratoladi.
  • PHP serverdagi fayllarni yaratishi, ochishi, o'qishi, yozishi, o'chirishi va yopoladi.
  • PHP formadagi ma'lumotlarni to'play oladi.
  • PHP kukilarni yuborishi va qabul qiloladi.
  • PHP sizning ma'lumotlar bazangizga ma'lumotlarni qo'shish, o'chirish, o'zgartiroladi.
  • PHP user-access(Foydalanuvchi kirishi) ni nazorat qiloladi.
  • PHP ma'lumotlarni shifrlay oladi.

Siz PHP'da faqat HTML bilan cheklanib qolmaysiz. Rasm, PDF fayllar qolaversa Flash-video larni veb sahifaga chiqarolasiz. Keyin har xil matn jumladan, XML va XHTML kabilar.

Nima uchun PHP?

  • PHP keng ma'lumotlar bazalarini qo'llab-quvvatlaydi
  • PHP turli xil platformalarda muammosiz ishlaydi. Jumladan: Windows, Linux, Unix, Mac OS X, v.h
  • PHP keng miqyosli ma'lumotlar omborini qo'llab quvvatlaydi.
  • PHP ni yuklab olish bepul. Uni rasmiy veb sahifasidan www.php.net yuklab olishingiz mumkin.
  • Serverda PHP ni yurgizish va uni o'rganish juda oson.
